## Job Logic Chi tiết

### GenerateMonthlyInvoices
```
Input: billingMonth (mặc định: tháng hiện tại)
Logic:
  FOR EACH tenant WHERE plan != expired AND is_active = true:
    SET search_path TO tenant_{slug}
    billing_day = settings['billing.auto'].auto_generate_day
    IF today.day != billing_day: SKIP (chỉ chạy đúng ngày)
    
    FOR EACH contract WHERE status = 'active':
      IF invoice already exists for this month: SKIP
      
      latest_reading = SELECT * FROM meter_readings
                       WHERE room_id = contract.room_id
                       ORDER BY reading_date DESC LIMIT 1
      
      prev_reading = SELECT * FROM meter_readings
                     WHERE room_id = contract.room_id
                       AND reading_date < billing_month
                     ORDER BY reading_date DESC LIMIT 1
      
      CREATE invoice with:
        electricity_old = prev_reading?.electricity_reading ?? 0
        electricity_new = latest_reading?.electricity_reading ?? 0
        water_old = prev_reading?.water_reading ?? 0
        water_new = latest_reading?.water_reading ?? 0
        is_auto_generated = true
      
      IF sendNotification = true:
        ENQUEUE SendInvoiceNotificationJob(invoiceId)
      
      LOG result
```

### SendInvoiceReminders
```
Logic:
  FOR EACH tenant:
    due_dates_to_remind = settings['billing.auto'].remind_before_days  # [3, 1]
    
    invoices = SELECT WHERE status IN ('pending')
               AND due_date - TODAY() IN (remind_days)
               AND payment_link_token IS NOT NULL
    
    FOR EACH invoice:
      IF not already sent reminder today:
        ENQUEUE ZaloReminderJob(invoiceId, 'upcoming')
        UPDATE notification_logs
```

### MarkOverdueInvoices
```
Logic: Midnight job
  UPDATE invoices SET status = 'overdue'
  WHERE status = 'pending' AND due_date < TODAY()
```

### RetryFailedNotifications
```
Logic: mỗi 15 phút
  notifications = SELECT * FROM notification_logs
                  WHERE status = 'failed' AND retry_count < 3
                  AND created_at > NOW() - INTERVAL '24 hours'
  
  FOR EACH notif:
    TRY resend via channel
    IF success: UPDATE status = 'sent', sent_at = NOW()
    ELSE: UPDATE retry_count += 1
          IF retry_count >= 3: UPDATE status = 'failed_permanent'
```

### RefreshZaloToken
```
Logic: mỗi ngày
  FOR EACH tenant WITH zalo.is_active = true:
    IF zalo_token expires in < 7 days:
      CALL Zalo API refresh token
      UPDATE settings['notification.zalo'].access_token
```

```csharp
app.Use(async (context, next) => {
  if (IsPublicEndpoint(context.Request.Path)) {
    await next(); return;
  }

  var tenantSlug = context.User.FindFirst("tenant_slug")?.Value;

  await next();
```

## File Upload Security
```
- Chấp nhận: image/jpeg, image/png, image/webp, application/pdf
- Max size: 10MB per file
- Scan filename: sanitize, không cho path traversal
- Lưu vào R2 với random UUID filename (không dùng tên gốc)
- Presigned URL hết hạn sau 1 giờ
```

### File Upload Flow (Presigned URL)
```
1. Client: POST /files/presign { filename, contentType, size }
2. Server: validate → generate R2 presigned PUT URL → return { uploadUrl, fileKey }
3. Client: PUT {uploadUrl} với file binary (direct upload lên R2)
4. Client: gửi fileKey cho API liên quan (VD: PUT /rooms/{id} với imageUrl = fileKey)
5. Server: verify fileKey tồn tại trong R2 → lưu vào DB
```

## Payment Security
```
Webhook verify MoMo:
  rawSignature = "accessKey=...&amount=...&extraData=...&...&requestId=..."
  signature = HMAC-SHA256(rawSignature, secretKey).ToHex()
  so sánh với request.signature

Webhook verify VNPay:
  sortedParams = sort all vnp_ params alphabetically
  rawHash = join with "&"
  secureHash = HMAC-SHA512(rawHash, hashSecret).ToUpper()
  so sánh với vnp_SecureHash

Idempotency:
  key = "payment:idempotent:{provider}:{orderId}"
  TTL = 72 hours
  SET NX (chỉ set nếu chưa tồn tại)
```

```typescript
api.interceptors.response.use(
  res => res,
  async err => {
    if (err.response?.status === 401 && !err.config._retry) {
      err.config._retry = true
      const newToken = await refreshAccessToken()
      err.config.headers.Authorization = `Bearer ${newToken}`
      return api(err.config)
    }
    return Promise.reject(formatError(err))
```

## Realtime (SignalR)

```typescript
// src/lib/hooks/useNotifications.ts
export function useNotifications() {
  const { accessToken } = useAuthStore()
  const { addNotification } = useNotificationStore()

  useEffect(() => {
    const connection = new HubConnectionBuilder()
      .withUrl(`${process.env.NEXT_PUBLIC_SIGNALR_URL}`, {
        accessTokenFactory: () => accessToken!
      })
      .withAutomaticReconnect()
      .build()

    connection.on('NewNotification', addNotification)
    connection.on('InvoicePaid', (data) => {
      // Invalidate React Query cache
      queryClient.invalidateQueries({ queryKey: ['invoices'] })
      queryClient.invalidateQueries({ queryKey: ['dashboard'] })
    })

    connection.start()
    return () => { connection.stop() }
  }, [accessToken])
}
```

## Middleware (Auth guard)

```typescript
// src/middleware.ts
export function middleware(request: NextRequest) {
  const token = request.cookies.get('accessToken')?.value
  const isAuthRoute = request.nextUrl.pathname.startsWith('/(auth)')
  const isDashboard = request.nextUrl.pathname.startsWith('/(dashboard)')
    || request.nextUrl.pathname === '/'

  if (isDashboard && !token) {
    return NextResponse.redirect(new URL('/login', request.url))
  }

  if (isAuthRoute && token) {
    return NextResponse.redirect(new URL('/', request.url))
  }
}
```
